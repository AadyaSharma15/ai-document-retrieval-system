# AI Document Retrieval System

A Python-based desktop application developed during Hack-AI-Thon for document search and retrieval.

## Features

* Load PDF and Word documents
* Extract text from files
* Search documents using TF-IDF ranking
* View full document content
* Save and load chat history
* Browser-assisted query search

## Technologies Used

* Python
* Tkinter
* scikit-learn
* PyMuPDF
* python-docx

## Project Purpose

This project was developed as a prototype to support knowledge retrieval by ranking documents based on query relevance.
